关于燃料电池负极反应方程式的简便书写方式

2021-02-20
化学 电池

首先需要了解

CXHYOZ + (X + Y/4 - Z/2) O2 === XCO2 + Y/2 H2O

即符合 CXHYOZ 通式的物质（X,Y,Z 取值均为自然数）在氧气中燃烧的通式。

通过此通式配平得到的方程式各项系数不一定为整数，因而需要将其各项系数化为正整数，得到的 CXHYOZ 的系数记为 M 而 O2 的系数记为 N。

通过燃烧通式以及正极反应同时相减可得到以下负极反应方程式：

注明：因以碱性物质作为电解质的燃料电池产生的 CO2 会与之反应，故应相加二氧化碳与氢氧根反应通式消去。

得到如下通式

CXHYOZ - (4X + Y - 2Z)e- + (2X + Y/2 - Z)O2 === -XCO2 + Y/2 H2O   熔融氧化物
CXHYOZ - (4X + Y - 2Z)e- + (6X + Y - 2Z)OH- === -XCO32- + (3X + Y - 2Z)H2O   碱性电解质
CXHYOZ - (4X + Y - 2Z)e- + (4X + Y - 2Z)OH- === -XCO2 + (2X + Y - 2Z)H2O  酸性电解质
CXHYOZ - (4X + Y - 2Z)e- + (2X + Y/2 - Z)CO32- === (3X + Y/2 - Z)CO2 + Y/2 H2O   熔融碳酸盐

书写时，方程式内相关系数应该扩展相同倍数，而此倍数即为 M。

另外，电池的正极反应式也应扩展相同倍数，而此倍数即为 N。

最后附上正极反应式通式

O2 + 4H+ + 4e- = 2H2O  酸性电解质
O2 + 4e- + 2CO2 = 2CO32-  碱性电解质
O2 + 4e- = 2O2-      熔融氧化物
O2 + 2CO2 + 4e- = 2CO32- 熔融碳酸盐

注意：碱性电解质溶液为电解质溶液时记住具体情况具体分析，另外附上不与二氧化碳反应的情况：

CXHYOZ - (4X + Y - 2Z)e- + (4X + Y - 2Z)OH- === -XCO2 + (2X + Y - 2Z)H2O 
O2 + 4e- + 2H2O = 4OH-
